# CareerCompass AI 🚀

**One-Stop Personalized Career and Education Advisor**

CareerCompass AI is a premium, full-stack AI platform designed to empower students and fresh graduates with personalized career intelligence, interactive learning roadmaps, and real-time market salary data.

## 🌟 Key Features

- **AI-Powered Recommendations**: Assessment-driven matching for 50+ high-growth tech careers.
- **Skill Gap Analysis**: Visual comparison of your current skills vs. market requirements.
- **Interactive Roadmaps**: Step-by-step master plans with progress tracking.
- **Salary Market Index**: Real-time salary projections for Indian tech roles.
- **AI Advisor Chatbot**: 24/7 career guidance bot to answer your queries.
- **Curated Resources**: Best-in-class courses from Udemy, Coursera, edX, and freeCodeCamp.

## 🛠️ Tech Stack

- **Frontend**: React (Vite), React Router, Axios, Lucide Icons, Recharts, Framer Motion, React Hot Toast.
- **Backend**: Node.js, Express, SQLite (`better-sqlite3`), JWT, BcryptJS.
- **UI/UX**: Custom Midnight Indigo design system with Glassmorphism and Responsive Layouts.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Khushirathore10/careercompass-ai.git
   cd careercompass-ai
   ```

2. **Setup Backend**
   ```bash
   cd backend
   npm install
   npm start
   ```

3. **Setup Frontend**
   ```bash
   # In a new terminal
   npm install
   npm run dev
   ```

4. **Access the App**
   Navigate to `http://localhost:5173`

## 📂 Project Structure

- `src/`: React frontend source code. (Components, Pages, Context, Styles)
- `backend/`: Express server, SQLite database, and API routes.
- `public/`: Static assets for the frontend.
